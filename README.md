# 🏦 Bank Customer Churn Analysis Dashboard

## Project Overview

This project analyzes customer churn behavior in the banking sector using SQL, Python, Excel, and Power BI. The goal is to identify factors influencing customer attrition and provide actionable business insights.

## Tools Used

- SQL Server
- Python
- Excel
- Power BI
- GitHub

## Project Workflow

1. Data Cleaning
2. SQL Analysis
3. Python Exploratory Data Analysis
4. Dashboard Development
5. Business Insights
